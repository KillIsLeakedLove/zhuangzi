# 📚 庄子智慧

> 「吾生也有涯，而知也无涯」—— 探索道家哲学的现代之旅

## 🌟 项目简介

一个精心打造的庄子经典阅读网站，汇集内篇、外篇、杂篇共 **28 章**，以现代化的交互方式呈现两千年前的东方智慧。

## ✨ 特色功能

| 功能 | 描述 |
|------|------|
| 📖 原文译文对照 | 经典原文 + 白话译文，便于理解 |
| 💡 深度解读 | 每章配有现代意义阐释 |
| 🎯 核心启示 | 提炼 5 点 actionable 智慧 |
| 📜 历史用例 | 3 个经典历史人物故事佐证 |
| 🎨 优雅界面 | 暗黑/明亮双主题，沉浸阅读 |

## 📚 内容结构

### 内篇 7 章
《逍遥游》《齐物论》《养生主》《人间世》《德充符》《大宗师》《应帝王》

### 外篇 15 章
《天道》《天运》《秋水》《至乐》《达生》《山木》《田子方》《知北游》《寓言》《让王》《说剑》《盗跖》《渔父》《列御寇》《天下》

### 杂篇 6 章
《庚桑楚》《外物》《则阳》《列御寇》《天下》等

## 🛠️ 技术栈

- **前端**：原生 HTML5 + CSS3 + JavaScript
- **样式**：CSS 变量主题系统
- **数据**：模块化 JavaScript 数据文件
- **交互**：原生 DOM 操作，无框架依赖

## 🚀 快速开始

```bash
# 克隆项目
git clone <repository-url>

# 进入目录
cd zhuangzi

# 打开页面（任选其一）
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

或直接访问在线版本：[庄子智慧网站](https://your-domain.com)

## 📖 使用指南

1. **浏览章节** - 左侧目录快速跳转
2. **阅读原文** - 点击文段查看译文
3. **深度学习** - 阅读解读、启示、用例三部分
4. **切换主题** - 点击右上角 🌙/☀️ 切换暗黑/明亮模式

## 🎨 主题预览

| 明亮模式 | 暗黑模式 |
|---------|---------|
| 清新淡雅 | 沉浸护眼 |
| 适合日间 | 适合夜间 |

## 📝 内容标准

- ✅ 原文精选经典段落 10-18 段
- ✅ 译文准确流畅，符合现代汉语
- ✅ 解读深入浅出，阐发现代意义
- ✅ 启示提炼 5 点核心智慧
- ✅ 用例采用《史记》等史书人物典故
